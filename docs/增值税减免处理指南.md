# 小规模纳税人免征增值税处理指南（季度必做）

> 适用：笑达公司（小规模纳税人，征收率 1%）。
> 政策依据：现行小规模纳税人**季度销售额 ≤ 30 万元免征增值税**。

## 一、为什么要做这步

开票时做了**价税分离**记账：

```
借 银行存款        100,000.00   （含税全款）
贷 主营业务收入      99,009.90   （不含税收入）
贷 应交税费-应交增值税   990.10   （代收的增值税）
```

如果当季销售额没超 30 万，这 990.10 **免征**、不用真交，但它会一直挂在
「应交税费-应交增值税(2221.01)」里虚增负债、并使净利润被低估。
必须每季度做一笔**减免结转**把它转出。

## 二、每季度末的减免分录

判断：当季「不含税」销售额 ≤ 300,000 元 → 免征。

```
借 应交税费-应交增值税(2221.01)   当季增值税额
贷 营业外收入(5301)              当季增值税额
```

- 日期：记在**该季度内**（如季末最后一天），确保进入当季报表。
- 之后必须做**该月期末结转**，把营业外收入(5301) → 本年利润(3131)，
  否则资产负债表「未分配利润」不准、报表不平。

## 三、与三张报表的对应

| 报表 | 科目/行次 | 说明 |
|------|-----------|------|
| 利润表 | 营业外收入(行22, r22) ← 5301 | 系统已映射，自动取数 |
| 资产负债表 | 应交税费(行36, l36) | 减免后应回到 0 |
| 资产负债表 | 未分配利润(行51) | 因营业外收入增加而上升 |

## 四、2026 年已处理记录

| 季度 | 不含税收入 | 增值税 | 是否免征 | 减免凭证 | 结转 |
|------|-----------:|-------:|:--:|------|------|
| Q1 | 69,306.93 | 693.07 | ✅ | 记-2026-010 (2026-03-31) | 转-2026-03 |
| Q2 | 99,009.90 | 990.10 | ✅ | 记-2026-011 (2026-06-30) | 转-2026-06 |

处理后：应交增值税余额 = 0；营业外收入累计 1,683.17；上半年净利润 79,848.00。
Q1/Q2 季度报表已重算并重新锁定。

## 五、以后每季度提醒（不要遗忘！）

1. 季末确认当季「不含税」销售额是否 ≤ 30 万。
2. 若免征：做减免分录（借 2221.01 / 贷 5301），金额 = 当季已计提的增值税。
3. 做该月期末结转（系统「期末」页面）。
4. 重新生成并锁定该季度报表。
5. 若**超过 30 万**：不做减免，到申报期实际缴纳：借 2221.01 / 贷 银行存款。
