# 笑达财务记账系统 - 使用指南

## 侧边栏导航

| 按钮 | 功能 | 路径 |
|------|------|------|
| 首页 | 查看已有财务报表，导出XLS | `/` |
| 记账 | 录入记账凭证，查看科目表 | `/voucher/` |
| 账簿 | 科目余额表、明细账、总账 | `/ledger/balance` |
| 期末 | 损益结转、自动生成三表 | `/closing/carry-forward` |

---

## 第一步：查看科目表

1. 点侧边栏 **记账** → 进入凭证列表
2. 点右上角 **科目表** 按钮
3. 可以看到已预置的 24 个科目（资产/负债/权益/收入/费用五大类）

已预置的科目是专门为小规模咨询公司定制的，包含管理费用-工资、应交增值税等常用科目。

---

## 第二步：日常记账（录入凭证）

点 **记账** → **新建凭证**，进入凭证录入页面。

### 场景1：收到咨询服务收入 50,000 元

| 字段 | 填写 |
|------|------|
| 日期 | 2026-05-05 |
| 摘要 | 收到XX公司咨询服务费 |
| 制单人 | 你的名字 |

分录填写（点"添加行"可增加行）：

| 科目 | 摘要 | 借方 | 贷方 |
|------|------|------|------|
| 1002 银行存款 | 收咨询费 | 50000.00 | |
| 5001 主营业务收入 | 咨询收入(不含税) | | 49504.95 |
| 2221.01 应交税费-应交增值税 | 增值税1% | | 495.05 |

> 价税分离：50000 ÷ 1.01 = 49504.95（不含税收入），增值税 = 495.05

底部会实时显示借贷合计和平衡状态，必须显示绿色"平衡"才能保存。

### 场景2：计提当月顾问工资 5,000 元

| 科目 | 摘要 | 借方 | 贷方 |
|------|------|------|------|
| 5602.01 管理费用-工资 | 计提5月工资 | 5000.00 | |
| 2211 应付职工薪酬 | 计提5月工资 | | 5000.00 |

### 场景3：银行转账发放工资

| 科目 | 摘要 | 借方 | 贷方 |
|------|------|------|------|
| 2211 应付职工薪酬 | 发放5月工资 | 5000.00 | |
| 1002 银行存款 | 发放5月工资 | | 5000.00 |

> 工资5000，个税为0，不需要代扣分录。

### 场景4：支付办公费 200 元

| 科目 | 摘要 | 借方 | 贷方 |
|------|------|------|------|
| 5602.02 管理费用-办公费 | 购买办公用品 | 200.00 | |
| 1002 银行存款 | 购买办公用品 | | 200.00 |

## 第三步：查看账簿

点侧边栏 **账簿**，进入科目余额表页面，顶部有三个 Tab 可切换：

### 科目余额表
- 选择年度和月份范围 → 点查询
- 显示每个有发生额的科目的借方合计、贷方合计、余额
- 点科目名称可直接跳转到该科目的明细账

### 明细账
- 选择具体科目（如"1002 银行存款"） → 点查询
- 显示该科目每一笔分录的日期、凭证号、摘要、借贷金额、逐笔余额

### 总账
- 选择年度和月份范围
- 按科目分组，显示每月的借贷发生额汇总

---

## 第四步：月末结转

点侧边栏 **期末** → 期末结转

1. 选择年度和月份（如2026年5月） → 点查看
2. 系统自动列出当月所有收入类和费用类科目的发生额
3. 显示汇总：收入合计、费用合计、本期净利润
4. 点 **一键生成结转凭证** → 系统自动创建一张结转凭证，将所有损益科目余额转入"本年利润"

> 结转凭证编号格式：`转-2026-05`，每月只能结转一次（防重复）。

---

## 第五步：自动生成报表

点 **期末** → 顶部 Tab 切换到 **生成报表**

1. 选择年度和季度（如2026年Q2） → 点预览
2. 系统从账簿数据自动计算三张报表：
   - **资产负债表**：货币资金、应收账款、应交税费、实收资本、未分配利润等
   - **利润表**：营业收入、管理费用、营业利润、净利润等
   - **现金流量表**：销售收到的现金、支付工资、支付税费等
3. 确认数据无误后，点 **确认生成报表**
4. 自动跳转到报表预览页 → 可以导出XLS

---

## 完整月度工作流（每月重复）

```
① 日常发生业务 → 记账（/voucher/new）录入凭证
② 月中随时查看 → 账簿（/ledger/balance）检查各科目余额
③ 月末         → 期末结转（/closing/carry-forward）一键结转损益
④ 季末         → 生成报表（/closing/generate-report）自动出三表
⑤ 报税         → 首页导出XLS → 上传税务系统
```

## 个税相关说明（退休返聘人员）

### 用工关系
- 退休人员不能建立劳动关系，签订《退休返聘协议》或《劳务协议》
- 不需要缴纳五险一金，建议购买商业意外险/雇主责任险

### 个税代扣代缴
- 月薪 5,000 元：应纳税所得额 = 5000 - 5000 = 0，个税为 0
- 公司仍需在自然人电子税务局（扣缴端）每月申报（零税额申报）
- 必须通过银行转账支付，保留凭证

## 技术信息

- 框架：Flask 3.1.3 + SQLite
- 启动：`cd xiaoda-tax && source venv/bin/activate && flask run --port=5001`
- 数据库：`instance/xiaoda.db`
- 科目初始化：`python seed_accounts.py`（仅首次需要）
