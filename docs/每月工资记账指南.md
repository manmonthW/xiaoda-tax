# 每月工资记账凭证录入指南

> 适用场景：退休返聘人员月薪 9,000 元，代扣个税 120 元，实发 8,880 元
> 每月需录入 3 张凭证，按时间顺序操作

---

## 每月时间线

```
月末（如4月30日）     → 录入凭证1：计提工资
次月发薪日（如5月10日）→ 录入凭证2：发放工资
次月15日前            → 在扣缴端申报个税并缴款
缴款当日              → 录入凭证3：缴纳个税
```

---

## 凭证1：计提工资

> **时间：** 每月月末（工资所属月份的最后一天）
> **含义：** 把本月的工资费用"记上账"，但还没有实际支付

### 操作步骤

1. 打开系统 → 侧边栏点 **记账** → **新建凭证**
2. 填写凭证头信息：

| 字段 | 填写内容 | 示例 |
|------|----------|------|
| 凭证编号 | 系统自动生成 | 记-2026-004 |
| 日期 | 工资所属月份最后一天 | 2026-04-30 |
| 摘要 | 计提X月工资 | 计提4月工资 |
| 制单人 | 你的名字 | |

3. 填写分录（3行）：

| 行 | 科目 | 摘要 | 借方 | 贷方 |
|:---:|------|------|---:|---:|
| 1 | **5602.01 管理费用-工资** | 计提4月顾问工资 | **9,000.00** | |
| 2 | **2211 应付职工薪酬** | 应付工资 | | **8,880.00** |
| 3 | **2221.02 应交税费-应交个人所得税** | 代扣个税 | | **120.00** |

4. 确认底部显示：
   - 借方合计：9,000.00
   - 贷方合计：9,000.00
   - 状态：**平衡** ✅
5. 点 **保存凭证**

### 理解这张凭证

```
借：管理费用-工资  9,000    ← 公司的费用增加了9000（利润减少）
  贷：应付职工薪酬  8,880   ← 公司欠员工8880（要付给他的钱）
  贷：应交税费-个税    120   ← 公司欠税务局120（替员工代扣的税）
```

---

## 凭证2：发放工资

> **时间：** 实际银行转账发薪的那一天
> **含义：** 钱从银行账户转出，付给了员工

### 操作步骤

1. **新建凭证**
2. 填写凭证头信息：

| 字段 | 填写内容 | 示例 |
|------|----------|------|
| 日期 | 实际发放日 | 2026-05-10 |
| 摘要 | 发放X月工资 | 发放4月工资 |

3. 填写分录（2行）：

| 行 | 科目 | 摘要 | 借方 | 贷方 |
|:---:|------|------|---:|---:|
| 1 | **2211 应付职工薪酬** | 发放4月工资 | **8,880.00** | |
| 2 | **1002 银行存款** | 发放4月工资 | | **8,880.00** |

4. 确认平衡 → **保存凭证**

### 理解这张凭证

```
借：应付职工薪酬  8,880    ← "欠员工的钱"清零了（负债减少）
  贷：银行存款    8,880    ← 银行里的钱少了（资产减少）
```

> 注意金额是 **8,880**（实发），不是 9,000。差额 120 是个税，走凭证3。

---

## 凭证3：缴纳个税

> **时间：** 在扣缴端申报并缴款成功的那一天
> **含义：** 把代扣的个税交给税务局

### 操作步骤

1. 先完成个税申报：
   - 打开自然人电子税务局（扣缴端）
   - 申报当月个税 → 缴款成功
   - 获取缴款凭证/截图
2. **新建凭证**
3. 填写凭证头信息：

| 字段 | 填写内容 | 示例 |
|------|----------|------|
| 日期 | 实际缴款日 | 2026-05-14 |
| 摘要 | 缴纳X月个人所得税 | 缴纳4月个人所得税 |

4. 填写分录（2行）：

| 行 | 科目 | 摘要 | 借方 | 贷方 |
|:---:|------|------|---:|---:|
| 1 | **2221.02 应交税费-应交个人所得税** | 缴纳4月个税 | **120.00** | |
| 2 | **1002 银行存款** | 缴纳4月个税 | | **120.00** |

5. 确认平衡 → **保存凭证**

### 理解这张凭证

```
借：应交税费-应交个税  120    ← "欠税务局的税"清零了（负债减少）
  贷：银行存款        120    ← 银行里的钱又少了120
```

---

## 三张凭证的完整资金流向

```
凭证1（计提）：
  管理费用 +9000  →  欠员工 8880 + 欠税局 120

凭证2（发薪）：
  银行 -8880  →  还清欠员工的 8880

凭证3（缴税）：
  银行 -120   →  还清欠税局的 120

最终效果：
  银行存款减少 9000（8880 + 120）
  管理费用增加 9000
  应付职工薪酬 = 0（已清）
  应交个税 = 0（已清）
```

---

## 每月科目余额变化验证

3张凭证全部录入后，到 **账簿 → 科目余额表** 检查：

| 科目 | 当月借方 | 当月贷方 | 期末余额 | 正确？ |
|------|---:|---:|---:|:---:|
| 1002 银行存款 | 0 | 9,000 | 减少9000 | ✅ |
| 2211 应付职工薪酬 | 8,880 | 8,880 | 0 | ✅ |
| 2221.02 应交税费-个税 | 120 | 120 | 0 | ✅ |
| 5602.01 管理费用-工资 | 9,000 | 0 | 9,000(借) | ✅ |

> 应付职工薪酬和应交个税余额为0，说明该付的都付了，该缴的都缴了。

---

## 附件管理建议

每张凭证应附以下原始凭证：

| 凭证 | 应附附件 |
|------|----------|
| 凭证1（计提） | 工资表（签字版） |
| 凭证2（发薪） | 银行代发工资回单 |
| 凭证3（缴税） | 个税扣缴端缴款截图/完税凭证 |

---

## 特殊情况处理

### 如果某月忘记计提，次月补录

在次月初补录凭证1，日期填回上月最后一天即可。系统按凭证日期归属月份。

### 如果个税缴款跨月

没关系，凭证3按实际缴款日期录入。月末 2221.02 科目可能会有 120 元贷方余额（代扣了但还没缴），属于正常，次月缴款后清零。

### 如果调整工资标准

修改凭证1中管理费用和应付职工薪酬的金额即可。个税根据新工资在扣缴端重新计算，凭证3金额相应调整。
