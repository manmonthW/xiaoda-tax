# 小规模纳税人首次雇佣退休返聘人员 — 企业管理全流程注意事项

> 适用场景：笑达公司（小规模纳税人），首次雇佣一名70+退休返聘人员，月薪9,000元
> 编写日期：2026年5月

---

## 一、银行代发工资

### 1.1 开通企业网银代发功能

| 步骤 | 操作 | 说明 |
|------|------|------|
| 1 | 携带公章、法人身份证去开户行 | 申请开通"代发工资"功能 |
| 2 | 签订《代发工资协议》 | 银行提供模板，约定代发方式和手续费 |
| 3 | 获取代发工资模板 | Excel格式，每月填写后上传网银 |
| 4 | 绑定员工收款账户 | 录入员工姓名、银行卡号、开户行 |

### 1.2 每月代发操作

1. 登录企业网银 → 代发工资模块
2. 上传/填写工资明细：
   - 员工姓名
   - 银行卡号
   - **实发金额：8,880 元**（9,000 - 120 个税）
3. 提交审批 → 复核（如有双人审批要求）
4. 确认发放 → 资金从对公账户划转至员工个人账户

### 1.3 注意事项

- **必须用对公账户转账**，绝不能用私人账户/现金发放
  - 私人转账 → 无法作为企业费用税前扣除
  - 现金发放 → 无法证明支付事实，税务检查时无法自证
- **保留银行回单**：每月打印或下载代发工资的银行回单，作为记账凭证附件
- **发放时间要固定**：建议每月固定日期（如每月5日或10日），形成规范

## 二、工资表与工资条

### 2.1 每月制作工资表（企业留存）

建议用 Excel 制作，包含以下字段：

| 序号 | 姓名 | 身份证号 | 应发工资 | 代扣个税 | 实发工资 | 发放日期 | 银行卡号 | 备注 |
|:---:|------|----------|---:|---:|---:|------|----------|------|
| 1 | XXX | 1301XXXX... | 9,000.00 | 120.00 | 8,880.00 | 2026-05-10 | 62XXXX... | 退休返聘 |

- 制表人签字、审批人签字
- 打印留存，归档保管（至少保存15年）

### 2.2 工资条（给员工）

每月向员工提供工资明细，员工签字确认：

```
笑达公司 工资条
───────────────────────────────
姓名：XXX            月份：2026年4月
应发工资：  9,000.00
代扣个人所得税：-120.00
实发工资：  8,880.00
发放日期：2026-05-10
───────────────────────────────
员工签字：________  日期：________
```

> 工资条不是法律强制要求，但强烈建议，可以避免日后纠纷。

## 三、合同/协议管理

### 3.1 必须签订的文件

| 文件 | 要点 | 份数 |
|------|------|:---:|
| **退休返聘协议** | 明确工作内容、报酬、期限、解除条件、保密条款 | 2份（各执一份） |
| **保密协议**（如涉及） | 如接触公司商业信息 | 2份 |

### 3.2 协议中必须写明的条款

- 用工性质：**劳务关系**，非劳动关系
- 报酬标准：月薪 9,000 元，税前
- 个税承担：**企业代扣代缴**，员工承担税款
- 工作时间：灵活/固定（建议写明）
- 协议期限：建议一年，自动续签条款
- 意外伤害：明确由商业保险覆盖，不适用工伤保险
- 解除方式：提前30天书面通知

### 3.3 不要签的文件

- ❌ 《劳动合同》— 退休人员无法建立劳动关系
- ❌ 社保相关文件 — 退休人员已退出社保体系

## 四、商业保险（必须办）

### 4.1 为什么必须买

退休返聘人员**不适用工伤保险**。如果工作中受伤：
- 没有工伤认定
- 公司承担全部民事赔偿责任
- 70+岁老人受伤赔偿金额可能极高

### 4.2 应该买什么险

| 保险 | 作用 | 年费参考 |
|------|------|----------|
| **雇主责任险** | 员工工作中受伤/死亡，保险公司赔付 | 200-500元/人/年 |
| **团体意外险** | 意外伤害/医疗，覆盖工作和非工作场景 | 100-300元/人/年 |

建议**两个都买**，年费合计不超过 800 元，但能转移数十万甚至上百万的风险。

### 4.3 投保注意

- 被保险人年龄上限：确认保险公司接受70+岁投保（部分产品上限65岁）
- 选择不限年龄或上限80岁的产品
- 投保时如实告知年龄和健康状况

## 五、企业所得税相关

### 5.1 工资支出税前扣除

退休返聘人员工资可以作为企业**管理费用**在企业所得税前扣除，但需满足：

| 条件 | 你的情况 |
|------|----------|
| 有合法的雇佣协议 | ✅ 退休返聘协议 |
| 有工资表 | ✅ 每月制作 |
| 通过银行转账支付 | ✅ 企业网银代发 |
| 已申报个人所得税 | ✅ 每月扣缴端申报 |

> 四个条件都满足 → 每月 9,000 元全额可以税前扣除（**不需要发票**）

### 5.2 商业保险费扣除

- 雇主责任险 → 可以税前扣除（列入管理费用-保险费）
- 团体意外险 → 参照当地政策，通常可以扣除

### 5.3 年度企业所得税影响

```
全年增加费用：
  工资支出：9,000 × 9个月 = 81,000 元
  保险费用：约 800 元
  合计增加费用：约 81,800 元

  → 减少应纳税所得额 81,800 元
  → 小微企业所得税率 5%，少缴企业所得税约 4,090 元
```

## 六、财务报表影响

### 6.1 资产负债表变化

| 科目 | 变化方向 | 说明 |
|------|----------|------|
| 银行存款（资产） | ↓ 减少 | 发放工资和缴税支出 |
| 应付职工薪酬（负债） | 先↑后↓ | 计提时增加，发放时减少，月末应为0 |
| 应交税费（负债） | 先↑后↓ | 计提时增加，缴款时减少 |
| 未分配利润（权益） | ↓ 减少 | 费用增加导致利润减少 |

### 6.2 利润表变化

| 项目 | 月度增加 | 说明 |
|------|----------|------|
| 管理费用 | +9,000 | 工资计入管理费用 |
| 利润总额 | -9,000 | 费用增加利润减少 |
| 所得税费用 | -450 | 利润减少所得税也减少 |
| 净利润 | -8,550 | |

### 6.3 现金流量表变化

| 项目 | 月度增加 |
|------|----------|
| 支付给职工的现金 | +8,880 |
| 支付的各项税费 | +120（个税）|

### 6.4 季度报表生成

在记账系统中的正确操作顺序：

```
1. 每月录入凭证（计提工资、发放工资、缴纳个税）
2. 每月做期末结转（/closing/carry-forward）
3. 季末自动生成报表（/closing/generate-report）
4. 核对三表数据 → 导出XLS → 用于报税
```

## 七、增值税申报影响

### 7.1 工资支出与增值税无关

- 发工资不涉及增值税
- 增值税只与营业收入有关
- 你的增值税申报不受雇人影响

### 7.2 季度增值税申报不变

- 仍按照季度收入正常申报
- 季度不含税收入 ≤ 30 万 → 免征增值税（政策不变）

## 八、每月/每季/每年必做事项清单

### 每月必做

- [ ] **月初~10日**：制作上月工资表 → 银行代发工资（实发8,880元）
- [ ] **15日前**：自然人电子税务局（扣缴端）申报个税 → 缴纳120元
- [ ] **发放当日**：在记账系统录入凭证（计提+发放+缴税共3张）
- [ ] **月末**：期末结转损益（系统一键操作）

### 每季必做

- [ ] 季末月的月末结转完成后 → 自动生成季度报表
- [ ] 核对三表数据
- [ ] 导出XLS → 用于季度企业所得税预缴申报
- [ ] 增值税季度申报（与工资无关，但要同期完成）

### 每年必做

- [ ] **1月**：申报12月个税
- [ ] **3月前**：确认员工是否需要年度汇算清缴（月薪9000不需要）
- [ ] **5月31日前**：企业所得税年度汇算清缴
- [ ] **续签或更新**退休返聘协议
- [ ] **续保**商业意外险/雇主责任险
- [ ] 整理全年工资表、银行回单、个税申报记录，装订归档

## 十、风险提示汇总

| 风险 | 后果 | 防范措施 |
|------|------|----------|
| 未签书面协议 | 劳动仲裁/民事诉讼时不利 | 入职前必须签退休返聘协议 |
| 用现金/私户发工资 | 无法税前扣除，无法证明支付 | 必须对公账户银行转账 |
| 未按月申报个税 | 罚款（税款0.5~3倍）+ 滞纳金 | 每月15日前必须申报 |
| 未代扣个税 | 公司作为扣缴义务人被处罚 | 发工资时扣税，实发8,880 |
| 未买商业保险 | 工伤赔偿全部由公司承担 | 购买雇主责任险+意外险 |
| 协议到期未续签 | 法律关系不清 | 提前30天续签或终止 |
| 工资表未签字 | 无法证明员工已领取 | 每月工资条签字确认 |
| 档案未保管 | 税务检查时无法提供凭证 | 纸质+电子双备份，保存15年 |
